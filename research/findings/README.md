# research/findings/

Gemini drops research findings here as markdown files.

**Filename format:** `YYYY-MM-DD-<task-id>-<slug>.md`  
Examples: `2026-03-16-A1-dominion-gs4-rates.md`, `2026-03-16-B1-440-monticello.md`

Claude reads these files, fact-flags, and promotes verified content to `blog/` or `briefs/`.

See `research/FINDING_TEMPLATE.md` for the required format.
